# Job Search Portal

A responsive front-end web application for exploring job opportunities. Users can browse available jobs and filter listings based on keywords through a simple and intuitive interface. The project is built entirely using client-side technologies without a backend. :contentReference[oaicite:0]{index=0}

## Features

- Browse job listings
- Search and filter jobs
- Responsive user interface
- Job posting page (UI)
- Login and Contact pages

## Tech Stack

- HTML5
- CSS3
- JavaScript
